The Pinfold App is developed using Flutter and Dart. It includes OSS packages from the pub.dev ecosystem and the Flutter SDK, which are subject to the BSD-style licenses and other terms published by the Flutter authors and individual package maintainers.

Representative categories of OSS used in the mobile App include:

  • Flutter SDK and Dart runtime libraries

  • Local storage, SQLite, and file system utilities
  • Authentication and OAuth client libraries
  • Media processing, image caching, and thumbnail generation tools
  • Cryptography and secure storage helpers
  • Testing, linting, and build tooling (development only)

3. Backend services (.NET)

Pinfold backend services are built on Microsoft .NET and incorporate OSS NuGet packages under licenses such as MIT, Apache License 2.0, and BSD.

Representative categories of OSS used in backend services include:

  • ASP.NET Core web framework and middleware
  • Entity Framework Core and database providers
  • JSON serialization, logging, and dependency injection libraries
  • HTTP clients and cloud SDK helpers
  • Security, JWT, and cryptography packages
  • Testing and code analysis tools (development only)
