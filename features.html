App features

Pinfold (com.pinfoldai.fileexplorer) groups ~48 capabilities across eight categories.

File management

Browse & search

Navigate folders, search by name, filter by type (images, videos, audio, documents).

File operations

Create, rename, delete, copy, cut, paste, multi-select, open, and share.

Archives

Create ZIP archives; extract ZIP and other supported formats.

Favorites

Star important files and folders for quick access.

Voice commands

Open Downloads, show videos, search by voice from the Files tab.

Security

App lock

PIN and optional biometric unlock for the whole app with auto-lock timeout.

Folder lock

Protect folders with PIN, biometric, or both.

Encrypted vault

Move sensitive files into an encrypted vault with your passphrase.

Recycle bin

Deleted files go to recycle bin first — restore or empty permanently.

Storage insights

Breakdown

Visual breakdown by file type on the Storage tab.

Large files

Find the biggest items consuming space on your device.

Category shortcuts

Jump from Storage into Files filtered by media type.

Productivity (Pro mode)

Duplicate finder

Scan for duplicate files and remove copies to free space.

Smart assistant

Health scan and cleanup suggestions with optional premium AI features.

WhatsApp cleaner

Scan and clean WhatsApp-related storage safely.

Global search

Indexed search across allowed storage locations.

Auto-organize

Rules to sort and move files automatically.

Cloud & backup

Pinfold Cloud

Account storage quota on Home (5 GB free tier; more with subscription).

Provider connect

OAuth connect to Google Drive, OneDrive, and Dropbox.

Sync queue

Background sync jobs with reliability center.

Migration Center

Advanced multi-provider cloud hub (premium).

Social Publishing Hub

Composer

Pick media, write captions, optional AI captions (premium), publish now.

Queue & calendar

Pending posts and scheduled content view (scheduling is premium).

Accounts

Connect Facebook, Instagram, Threads, X, TikTok, LinkedIn, YouTube, and more.

Analytics

Publishing dashboard (premium).
